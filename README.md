## Features

- able to visualize very long axis.
- efficient for periodic data.

## Install

If you want the latest version, install it directly from GitHub:

```r
library(devtools)
install_github("jokergoo/spiral")
```

## Usage

```r
library(spiral)
spiral_initialize()
spiral_track()
spiral_points()
```
